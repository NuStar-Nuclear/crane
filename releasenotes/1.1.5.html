What’s New in 1.1.5¶

New Features¶

  1. 增加各种组件布置图的可视化功能；

  2. 增加对称性是否满足的自动检查逻辑；

  3. 增加每个组件的轴向功率分布可视化功能；

  4. 增加能群平均的Dancoff因子可视化功能；

Bug Fixes¶

  1. 7fa3125: NAN should not be annotated when use nonzeros comparation

  2. 0da163f: incorrect dancoff factor edited if resonance region is divided to mutli cells

  3. c2d238a: invalid mesh generated by unfilled region geos intersecting with mesh geos

  4. 916bce4: DIRECT model bug, use steady conduction

  5. 1bc86fa: modify MIN_CELL_SIDE_LENGTH 0.05 -> 1e-6 to fix tiny triangle cell bug for cmfd

  6. 613d457: MAX_NUM_SIDE_PER_CELL = 8 is not enough for same hexagonal cases, so set it 10

  7. dc32b3e: pad_map error when padding hexgonal map

  8. 7745aae: thread not safe to insert a global variable

  9. a755ce6: modified Cooltable & fluidprop bug

  10. 5b6035e: add coolprop table routines, and modified pip bug

  11. 9df1478: incorrect statepoint time statics

  12. 6159e5f: _z_mesh_bot_fraction computation error, which cause bad axial offset

  13. e4696fe: peak value patch color not consist with real peak value color

  14. f69d677: if openmc-denpent lib not installed, OSError raised

  15. 880fc7c: add test Yaml and repair bug

  16. edabce4: incorrect day time point calculated if power ratio less than 1.0

  17. 2ac9103: burnup point now do not contain 0.0

  18. f0cabc8: library reading error when number of nuclides changed

  19. 3e99e5c: format error

  20. 29b4f12: critical boron can not search down to zero

  21. 54e2fb5: printing head not align

  22. 134fa38: all-nan error when none interesting data found

  23. c2d7c40: bugs accurred when is_plane_xs_generator=true

  24. dc6a0d0: core symmetry should be token account in total heavy metal accumulation

  25. f0055de: moc mesh flux key error
